# 解耦执行 / unstuck

一个手机上能用的小工具，专门用来对付"明知道该做什么，但就是不想动"的那种卡住时刻。

**在线打开：** https://slowrecover.github.io/unstuck/

## 这是干嘛用的

心情不对、犯懒、逃避一件事的时候，脑子里常常是这样的：知道要做，但情绪上完全不想动，越想越拖。

这个工具不讲道理、不打鸡血，就是带你走五步，把"卡住"拆解掉：

1. 说清楚你现在具体是什么情绪（不是"不好"，是烦躁？恐惧？无力？）
2. 说出这个情绪最想让你干嘛（比如刷手机），然后反着来一次
3. 把要做的事压缩成一个两分钟内就能开始的最小动作
4. 用"我选择"造句，自己确认一下（不用保证做到，说出口就行）
5. 生成一行记录，方便你复制粘贴留档

全程不联网、不用登录、不会有人看到你写了什么——所有内容只存在你自己手机的浏览器里。

## 怎么用

1. 手机浏览器打开上面那个网址
2. 浏览器菜单里找"添加到主屏幕"，加个图标，以后像个小 App 一样点开用
3. 卡住的时候点开，跟着五步走一遍就行

## 这个东西的依据

不是瞎编的，背后是几个真实的心理学研究方向：

- **具体命名情绪能降低情绪强度**（affect labeling，Lieberman 的研究）
- **不压抑冲动，只是不服从它**（DBT 的 opposite action 技术）
- **行动可以先于动机，不用等感觉对了再做**（行为激活疗法）
- **用"我选择"而不是"我必须"，自主感能提高行动的依从性**（自我决定论，Deci & Ryan）

## 说明

- 纯前端页面，没有后端、没有数据库、不发任何网络请求
- 记录只存在浏览器的 localStorage 里，换设备/清缓存就没了，请自己复制保存
- 这是个人在用的小工具，不追求功能齐全，能解决"卡住"这一个问题就够了
